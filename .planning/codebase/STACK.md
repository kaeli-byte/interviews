# Tech Stack

## Context
Tech stack overview for MyCareer AI app.

## Core Runtime
- **Node.js**
- **Next.js 16.2.0** (App Router)
- **React 19.2.4**
- **TypeScript 5**

## Frontend
- **Tailwind CSS v4** for styling
- **Shadcn** & **Base UI** (@base-ui/react ^1.3.0) for UI primitives
- **Framer Motion** for animations
- **Lucide React** for icons
- **class-variance-authority** & **clsx** & **tailwind-merge** for class utilities

## Tooling
- **ESlint 9**
- **TypeScript**

## Status
Modern frontend stack with cutting edge Next.js and Tailwind CSS versions.
