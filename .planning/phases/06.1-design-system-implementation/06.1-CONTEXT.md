#### Color Palette
- **D-01:** Primary Colors — Primary (#003f98), Secondary (#4c616c), Tertiary (#004d4d), On-surface (#191c1d)
- **D-02:** Glass Tint Palette — `glass-tint-dark` (rgba(0,0,0,0.20)), `glass-tint-light` (rgba(255,255,255,0.025)), `glass-border` (rgba(255,255,255,0.50)), `glass-highlight` (rgba(255,255,255,0.75))

#### Surface System
- **D-04:** Tonal Surface Hierarchy — surface, surface-container-low/lowest/high (for non-glass surfaces)
- **D-05:** Glass Surface Hierarchy — `.glass` (light tint), `.glass-panel` (dark tint), `.glass-shadow` (shadow stack)

#### Liquid Glass Effects
- **D-06:** Glass Shadow Stack — `inset 0 1px 0px rgba(255,255,255,0.75), 0 0 9px rgba(0,0,0,0.2), 0 3px 8px rgba(0,0,0,0.15)`
- **D-07:** Gradient Overlays — `before::` gradient (top-left to bottom-right, 60% white) + `after::` gradient (bottom-left to top-left, 30% white)
- **D-08:** Glass Interactions — `hover:bg-white/30` with `transition-all duration-300`

#### Typography
- **D-09:** Font Stack — Manrope (display/headlines) + Inter (body/transcripts)
- **D-10:** Text on Glass — White text with opacity variations (100%, 70%, 60%)

#### Utilities
- **D-11:** Utility Classes — `.glass`, `.glass-panel`, `.glass-shadow`, `.glass-shine`, `.editorial-gradient`, `.shadow-ambient`, `.ghost-border`
