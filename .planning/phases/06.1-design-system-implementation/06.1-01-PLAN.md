<!-- Design spec from DESIGN.md Liquid Glass Edition - these values are the source of truth -->

**Primary Colors (D-01):**
- Primary: #003f98 (deep authoritative blue)
- Primary-Container: #1a56be (lighter blue for gradients)
- Secondary: #4c616c (sophisticated grey)
- Tertiary: #004d4d (growth-oriented teal)
- On-Surface: #191c1d (for shadow tinting)

**Glass Tint Palette (D-02):**
- glass-tint-dark: rgba(0,0,0,0.20) — Glass panels over light backgrounds
- glass-tint-light: rgba(255,255,255,0.025) — Glass buttons, subtle surfaces
- glass-border: rgba(255,255,255,0.50) — Semi-transparent white borders
- glass-highlight: rgba(255,255,255,0.75) — Inner top-edge shine

**Fixed Variants (D-03):**
- primary-fixed: oklch(0.70 0.14 250) — High-visibility accent
- primary-fixed-dim: oklch(0.75 0.12 250) — Muted accent
- tertiary-fixed: oklch(0.60 0.12 180) — Tertiary accent on dark

**Surface Hierarchy (D-04):**
- surface: #f8f9fa (base layer)
- surface-container-low: #f3f4f5 (sections)
- surface-container-lowest: #ffffff (active cards)
- surface-container-high: #e7e8e9 (nested controls)
- surface-container-highest: #dadbdc (deepest layer)

**Glass Shadow Stack (D-06):**
- Inner highlight: inset 0 1px 0px rgba(255,255,255,0.75)
- Soft glow: 0 0 9px rgba(0,0,0,0.2)
- Drop shadow: 0 3px 8px rgba(0,0,0,0.15)

**Gradient Overlays (D-07):**
- before:: gradient: to bottom-right, from white/60 via transparent to transparent, opacity 70%
- after:: gradient: to top-left, from white/30 via transparent to transparent, opacity 50%

**Typography (D-09):**
- Display/Headlines: Manrope (geometric, modern, editorial)
- Body/Transcripts: Inter (maximum legibility)

**Ambient Shadows (D-11):**
- Blur: 32-64px, Opacity: 4-8%, Tinted with on-surface (#191c1d)
