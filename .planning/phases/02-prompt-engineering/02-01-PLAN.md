  <action>
    Create `lib/promptBuilder.ts` with the system instruction builder utility.

    1. Define interface `PromptContext`:
       - `resume: string`
       - `jobDescription: string`
       - `personality: PersonalityKey` (import from personalities.ts)

    2. Define constant `MAX_CONTEXT_CHARS = 8000` for safe truncation (~2K tokens with buffer)

    3. Export function `buildSystemInstruction(ctx: PromptContext): string`:
       - Truncate resume and JD to MAX_CONTEXT_CHARS using `.slice(0, MAX_CONTEXT_CHARS)`
       - Import and lookup personality constraints: `const personality = PERSONALITY_PRESETS[ctx.personality]`
       - Return template literal with XML-delimited sections:
         - `<role>`: "You are an expert career coach and technical interviewer conducting a mock interview. Your goal is to help the candidate articulate their unique value proposition."
         - `<context>`: Contains `<resume>` and `<jobDescription>` subsections with truncated text
         - `<personality>`: The personality.instructions block
         - `<opening>`: "Begin with a warm, personalized icebreaker that references 1-2 specific details from the candidate's resume (e.g., their current role, a notable achievement, or a relevant skill mentioned above). After your icebreaker, ask your first interview question to begin the conversation."
         - `<constraints>`: "- Keep questions focused and allow candidate time to respond fully\n- Adapt follow-up questions based on candidate's answers\n- When time is ending (user says they're done), conclude gracefully"
       - Use `.trim()` to remove leading/trailing whitespace

    4. The function must be pure (no side effects, deterministic output for same input).
  </action>
  <verify>
